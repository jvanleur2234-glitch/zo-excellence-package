## Build Your Own

1. Do the task once manually in JackConnect
2. Watch Once captures it
3. Skill is generated automatically
4. Test it, improve it
5. Share it here

```bash
cd jack-connect/watch-once
python3 watch_once_loop.py init "Your task name"
python3 watch_once_loop.py create <session-id> <skill-name>
```
